Paperless AI Analyzer

Intelligent document intelligence for Paperless-ngx — anomaly detection, AI analysis, project management, and natural-language search.

🗂️ Projects

Organize documents into isolated workspaces (cases, clients, matters). Each project has its own AI vector store — chat and search stay scoped to the selected project.

📤 Smart Upload

Upload files, paste a URL, or share a Google Drive / Dropbox link. Assign the document to a project at upload time. Optional AI metadata extraction before submission.

🤖 AI Chat

Ask natural-language questions about any document or your entire project archive. RAG-powered answers cite the source document.

🔍 Search & Analysis

Semantic full-text search across all analyzed documents with AI-generated summaries, risk scores, and anomaly tags displayed inline.

⚠️ Anomaly Detection

Deterministic checks (balance mismatches, missing transactions, duplicate amounts) run automatically on every document without an AI call.

🔒 Multi-User

Role-based access (Basic / Admin). Each user has their own chat history and upload history. Admins manage accounts, projects, and system settings.

How it works

Paperless AI Analyzer runs as a sidecar microservice alongside Paperless-ngx. It polls Paperless every ~30 seconds, picks up newly-ingested documents, and runs them through a pipeline:

  1. Profile matching — the document is matched to a YAML analysis profile (e.g., "bank statements", "court documents").
  2. Project detection — if the document has a project:slug tag in Paperless, it is routed to that project's vector store collection.
  3. Deterministic checks — balance calculations, transaction counts, and pattern checks run without any AI call.
  4. AI analysis — OpenAI GPT-4o or Anthropic Claude analyzes the document and extracts metadata (entities, topics, summaries, Q&A pairs).
  5. Tagging — anomaly tags (e.g., anomaly:balance_mismatch) and AI tags are written back to Paperless.
  6. Embedding — the enriched content is stored in the correct Chroma vector collection for AI Chat retrieval.
Non-destructive by design. The analyzer only adds tags and writes to its own database. Paperless documents are never modified — only tagged. You can remove the analyzer at any time without affecting your Paperless archive.
