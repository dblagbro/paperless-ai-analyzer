User Management

Create, edit, and manage user accounts. Admin role required.

User management is in Configuration → Users and is visible only to Admin users.

User roles

RolePermissions
AdminFull access: manage users, view all chat sessions, change system settings, manage projects, access all configuration sub-tabs.
AdvancedAll Basic permissions plus: run Case Intelligence analyses, use Report Builder, access power-user features. Cannot access Configuration or manage other users.
BasicCan analyze documents, upload, search, use AI Chat (own sessions + shared), manage own profile. Cannot access Configuration or other users' data.

Role is assigned at user creation and can be changed by an Admin via Configuration → Users → Edit (pencil icon).

Creating a user

  1. Go to Configuration → Users.
  2. Click + Add User.
  3. Fill in the required fields: Username, Display Name, Email, Password, Role.
  4. Optionally fill the profile fields (phone, job title, etc.).
  5. Click Create User.

If SMTP is configured, a welcome email is automatically sent to the new user's email address with login instructions.

Editing a user

Click the Edit (pencil) icon next to any user in the table. The Edit dialog lets you change:

  • Display Name, Email, Role, Job Title — any time.
  • Password — leave blank to keep current password; enter a new value to reset it.
  • All profile fields (phone, address, social handles, timezone).

Deactivating a user

Click the Deactivate (trash) icon. The user account is soft-deleted — they can no longer log in, but their chat sessions and upload history are preserved. Deactivated users appear with a strikethrough in the users table. Click Reactivate to restore access.

You cannot deactivate your own account. At least one Admin account must remain active at all times.

My Profile

Any user can view and update their own profile: click the 👤 My Profile button in the top-right corner of the dashboard. The profile modal lets you edit:

  • Full Name — display name shown in the UI and email notifications.
  • Email — used for SMTP notifications, welcome emails, and CI run notification emails.
  • Phone, Address, Job Title — optional contact details stored in your profile.

To change your password, expand the Change Password section within the same modal — enter your current password, the new password, and confirm. Password and profile fields are saved separately.

Users cannot change their own username or role. Admins can change any user's role via Configuration → Users → Edit.
