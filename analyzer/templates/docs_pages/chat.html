AI Chat

Natural-language conversation grounded in your actual project documents using RAG (Retrieval Augmented Generation).

Requires AI key. AI Chat needs at least one OpenAI or Anthropic API key configured under Configuration → AI Settings. A Cohere key is needed for the vector store (RAG retrieval).

How RAG retrieval works

Every message you send triggers a two-step process:

  1. Retrieve — your query is embedded (Cohere) and compared semantically against all documents in the current project's vector store. The most relevant documents (up to 5–8) are retrieved.
  2. Generate — the retrieved document content is included in the AI's context window alongside your question. The AI generates an answer grounded in the actual document text — no hallucination of figures or names.

The AI is instructed to only report numbers and facts explicitly present in the retrieved documents. If a document doesn't contain the requested information, the AI will say so rather than guess.

Starting a session

  1. Click + New Session in the left sidebar.
  2. Type your first question and press Enter.
  3. The session title is auto-generated from your first message. You can rename it by clicking the pencil icon next to the title.

Web URL access

You can paste a URL directly into any chat message and the AI will fetch the page and use its content in its answer. This is useful for:

  • Comparing your documents to a published court decision or news article
  • Asking the AI to summarize an external case and relate it to your documents
  • Providing supplemental reference material (statute text, regulatory guidance, etc.) without uploading it

Up to 3 URLs per message are fetched. HTML formatting is stripped — only readable text is sent to the AI. Each URL is truncated to approximately 4,000 characters.

Limitations: Pages that rely on JavaScript to render (single-page apps, court dockets behind login walls) may return little or no readable text. In that case the AI will tell you it could not read the page and suggest you paste the text directly. The AI will never guess or fabricate content from a URL it could not access.

Vision AI fallback

If a retrieved document has very little OCR text (fewer than ~500 characters), the system automatically:

  1. Downloads the original PDF from Paperless.
  2. Sends PDF page images to GPT-4o Vision or Claude Vision for text extraction.
  3. Uses the extracted text for this answer AND re-embeds the document with the better content for future queries.

When Vision AI is used, the response includes a note indicating which documents were enriched this way. Treat Vision AI-extracted content as authoritative.

Filtering by document type

The Filter by Type dropdown above the chat input limits retrieval to a specific document category (e.g., "Bank Statement", "Court Order"). This is useful when you want answers only from a specific document class.

Editing past messages

Each of your own messages has a visible ✏️ Edit button to its left — no hovering required. Clicking it turns the bubble into an editable textarea (white background). You have two options:

  • ↩ Resend — saves the edited text, removes the old assistant response (and any messages after it), and sends the edited message as a new query. The conversation continues from the edit point.
  • ✕ Cancel — discards the edit and restores the original text.

The Edit button appears on both current-session messages and on messages restored from a saved session. Edited messages are persisted to the session — if you reload the page or restore the session, the edited text appears (not the original).

Stopping a request

A red ■ Stop button appears next to the Send button while the AI is generating a response. Click it to abort the request immediately. A "⚠️ Request stopped" notice replaces the typing indicator, the input is re-enabled instantly, and a ✏️ Edit button is attached to the stopped message so you can fix it and resend without retyping.

Dual LLM compare mode

Click the ⚖️ Compare LLMs button in the chat input bar to enable compare mode (the button turns purple and reads "⚖️ Comparing — ON"). In compare mode, each message is sent to both configured LLM providers simultaneously and the responses are displayed side-by-side as tabs.

  • Tabs are labelled with the provider name (e.g. 🤖 Anthropic | 🤖 Openai). Click to switch between them.
  • The primary provider's response is saved to the session history so follow-up questions remain coherent.
  • If the secondary provider errors, the tab shows the error message in red.
  • Requires both a primary and a fallback provider to be configured in ⚙️ Configuration → AI Settings.

Clickable document links

When the AI references a document using the format [Document #NNN], it is automatically converted to a clickable "View in Paperless" link that opens the document directly in your Paperless instance. The AI is instructed to always use this format, but the linkifier also catches informal variants like Doc 123.

Sharing sessions

  1. Open a session.
  2. Click Share (icon in the session header).
  3. Select one or more users from the dropdown. Click Save.

Shared sessions appear in the recipient's sidebar with a "Shared" label. Recipients can read the full conversation but cannot add messages to sessions they don't own.

Admins can see and manage all sessions across all users.

Exporting a session

Click Export PDF in the session header. The entire conversation history is downloaded as a formatted PDF suitable for case notes or discovery production.

Tips for effective queries

  • Be specific about the project — switch to the correct project before chatting to scope the context.
  • Reference time periods — "statements from Q1 2024", "orders issued before March 2025".
  • Ask for comparisons — "compare the ending balances in January and February statements".
  • Request structured output — "list all documents with balance mismatches in a table".
  • Follow up — sessions maintain history. You can ask follow-up questions that reference earlier answers.

Example queries

  • "What is the ending balance on the most recent Chase statement?"
  • "Which court orders mention Fairbridge Real Estate?"
  • "Summarize all documents filed in 2024."
  • "Are there any documents with balance mismatches? What are the discrepancy amounts?"
  • "List all vendors that appear across the invoices."
  • "Generate a ledger from all statements for account ending in 4521."
