🔧 CI Setup — Configure and launch an analysis run.
  • Role — Plaintiff / Defendant / Neutral — shapes how the Director frames analysis and theories. CI also generates opposing-side theories automatically so you see what you're up against.
  • Jurisdiction — choose a template or leave blank for auto-detect from documents.
  • Goal / Case Description — what to find. Click ✨ Refine with AI to chat with an assistant that helps craft a precise, effective goal.
  • Budget (USD) — per-run spending cap. Run pauses if exceeded; override button shown. More documents = proportionally more spend; budget is a ceiling, not a target.
  • Orchestration Tiers: Director = 1 (fixed, plans all work); Managers = Auto (min(6, docs÷20)) or 1–10; Workers/Manager = Auto (budget-scaled) or 1–20.
  • Progress bar: shows stage · docs processed · active managers/workers · tokens used · elapsed time · ETA (after 10% complete).
  • Notifications: Email on run complete + every 10% budget checkpoint. Recipient pre-filled from your 👤 My Profile email.
  • Click ▶ Start Analysis → Director generates clarifying questions → answer or check "Proceed with assumptions".

📋 CI Findings — Browse run results and generate reports.
  • Select a run from the dropdown; live progress shows while running (Cancel button available).
  • Run metadata header — who ran it, when, duration, role, document count, and AI cost.
  • 6 finding domains: Entities · Timeline · Financial · Contradictions · Theories · Authorities. Every finding cites source document + page. Theories include both your role's perspective and the opposing side's strongest arguments.
  • AI Chat integration: CI findings are embedded into the project's vector store after each run — ask AI Chat questions and it will cite CI findings alongside document excerpts.
  • 👥 Share — share a run with another Advanced or Admin user by username. Shared users can view all findings and download the report.
  • 🗑 Delete Run — permanently removes a completed run and all its findings.
  • Report Builder: Director synthesizes a 12-section scientific paper (Executive Summary → Discovery Gaps + Appendices A–C) → PDF download.
